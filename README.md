# ENTR-451 Assignment #4

Full instructions at https://entr451.com/assignment-4/

This application is a Travel Journal that allows users to record places they visited and add pictures, dates and descriptions of events they experienced in those places. 
Travel Journal is a Ruby on Rails web app that lets users sign up, log in, create their own places, and keep a personal travel log of entries for each place. Each entry includes a title, description, occurred-on date, and optional uploaded image, and users can only view and manage their own places and entries.
